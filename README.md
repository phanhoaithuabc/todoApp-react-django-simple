# todoApp-react-django

### Simple TODO app using Django as backend and ReactJS as frontend

#### To start the app:
1. cd to project folder and run:
    ```
    python manager.py runserver
    ```
2. cd to frontend folder and run:
    ```
    npm install
    npm start
    ```
